# XO Game (Tic Tac Toe)

## 📌 Overview

This is a simple **Tic Tac Toe (XO)** game built using **C# and Windows Forms**.
The game allows two players to play against each other on the same computer.

The program keeps track of turns, checks for a winner automatically, and shows the result when the game ends.

---

## 🎮 Game Rules

* The game is played by **two players**.
* **Player 1** uses **X**.
* **Player 2** uses **O**.
* Players take turns clicking on the grid.
* The first player to align **three symbols** (horizontally, vertically, or diagonally) wins.
* If all **9 cells** are filled with no winner, the result is a **Draw**.

---

## 🖥️ Features

* Turn indicator for players.
* Automatic winner detection.
* Draw detection when the board is full.
* Highlight winning cells.
* Restart game button.
* Error message when selecting an already used cell.
* Simple graphical grid using `Paint`.

## ▶️ How to Run

1. Open the project in **Visual Studio**.
2. Build the project.
3. Run the program.
4. Click **Start** to open the game.
5. Play by clicking on the grid buttons.

---

## 🔄 Restarting the Game

Press the **Restart** button to reset:

* Board cells
* Turn indicator
* Winner label
* Play count
